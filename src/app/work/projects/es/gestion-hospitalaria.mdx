## Descripción general

API REST en producción que alimenta un ERP hospitalario. Más de 80 endpoints construidos con ASP.NET Core 8 y Clean Architecture, sirviendo a todos los departamentos del hospital en uso productivo activo.

## Características principales

- **Entrega** de más de 80 endpoints REST cubriendo las operaciones centrales del hospital: ASP.NET Core 8 con Clean Architecture como columna vertebral del ERP en producción
- **Sistema** de logging por roles utilizado diariamente por soporte, eliminando las escalaciones de errores al equipo de desarrollo
- **Documentación** de reglas de negocio directamente con los jefes de departamento antes de escribir código, previniendo desvíos de requerimientos y reduciendo el retrabajo durante el desarrollo
- **Contratos** de API compartidos con el equipo móvil (Kotlin/Jetpack Compose), habilitando pistas de desarrollo en paralelo y reduciendo retrasos de integración
