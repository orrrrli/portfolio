## Overview

Production REST API powering a hospital management ERP. 80+ endpoints built with ASP.NET Core 8 and Clean Architecture, serving all hospital departments in active production use.

## Key Features

- **Delivered** 80+ REST endpoints covering core hospital operations: ASP.NET Core 8 with Clean Architecture as the backbone of the production ERP
- **Shipped** a role-based logging system used daily by support, eliminating error escalations to the development team
- **Documented** business rules directly from department heads before writing code, preventing requirement drift and reducing mid-development rework
- **Established** shared API contracts with the mobile team (Kotlin/Jetpack Compose), enabling parallel development tracks and reducing integration delays
