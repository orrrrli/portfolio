## Overview

Enterprise backend for a hospital management system, built using Clean Architecture and CQRS. The entire REST API powers the ERP used by the hospital's departments in production.

## Key Features

- **REST endpoints** covering core hospital operations built with ASP.NET Core 8
- **Role-based logging system** that lets support teams diagnose errors without escalating to development
- **Business rule documentation** gathered directly from department heads before writing code
- **API contracts coordinated** with a mobile team (Kotlin/Jetpack Compose) to allow parallel development

## Tech Stack

ASP.NET Core 8, C#, Clean Architecture, CQRS/MediatR, Entity Framework Core, PostgreSQL
